# Roblox Developer Portfolio Template

A modern, responsive portfolio template for Roblox developers.

Automatically fetches:
- Public games
- Roblox avatar
- Profile links

## Features

- Single file setup
- PHP based (no database required)
- TailwindCSS styling
- Auto game thumbnails
- Tech stack modal
- SEO ready
- Mobile responsive

## Requirements

- PHP hosting
- cURL enabled

## Installation

1. Upload `index.php` to your hosting
2. Open the file
3. Edit the configuration section at the top
4. Save & refresh

Done.
